# Images de la section "Nos Produits"

## Emplacement des images

Placez vos images personnelles dans ce dossier avec les noms suivants :

- `produit-1.jpg` (ou .jpeg, .png, .webp) - **Fruits secs**
- `produit-2.jpg` (ou .jpeg, .png, .webp) - **Riz & Céréales**
- `produit-3.jpg` (ou .jpeg, .png, .webp) - **Produits poudrés & Épices**
- `produit-4.jpg` (ou .jpeg, .png, .webp) - **Fruits**
- `produit-5.jpg` (ou .jpeg, .png, .webp) - **Produits alimentaires**
- `produit-6.jpg` (ou .jpeg, .png, .webp) - **Logistique & Douanes**

## Recommandations

- **Format** : JPG, JPEG, PNG ou WebP
- **Dimensions recommandées** : 600x400px (ratio 3:2)
- **Taille** : Optimisez vos images pour le web (< 200KB par image)
- **Qualité** : Images nettes et professionnelles

## Note

Les images seront automatiquement redimensionnées et adaptées au design existant. Le CSS garantit un affichage cohérent sur tous les appareils.
