# Images Immobilier

Ce dossier contient les images pour la page immobilier.html.

## Structure des images

### Villas & Appartements (3 images)
- `villa1.jpeg` - Première image du carrousel Villas
- `villa2.jpeg` - Deuxième image du carrousel Villas
- `villa3.jpeg` - Troisième image du carrousel Villas

### Gestion Locative (4 images)
- `GS1.jpeg` - Première image du carrousel Gestion Locative
- `GS2.jpeg` - Deuxième image du carrousel Gestion Locative
- `GS3.jpeg` - Troisième image du carrousel Gestion Locative
- `GS4.jpeg` - Quatrième image du carrousel Gestion Locative

## Instructions

1. Placez vos images dans ce dossier avec les noms exacts indiqués ci-dessus.
2. Formats supportés : `.jpg`, `.jpeg`, `.png`, `.webp`
3. Dimensions recommandées : 1200x800px (ratio 3:2)
4. Taille recommandée : < 500KB par image pour un chargement optimal

## Remplacement facile

Pour remplacer une image, copiez simplement votre nouveau fichier avec le même nom dans ce dossier.
