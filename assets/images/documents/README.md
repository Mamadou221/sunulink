# Images Documents

Ce dossier doit contenir les images de prévisualisation des documents PDF.

## Fichiers requis

- `catalogue.jpg` ou `catalogue.png` - Image de prévisualisation du catalogue produits
- `brochure.jpg` ou `brochure.png` - Image de prévisualisation de la brochure de présentation

## Note

Si les images ne sont pas disponibles, le code affichera automatiquement une icône SVG de document en remplacement.
