# Logo et Favicon

Ce dossier contient les fichiers du logo et du favicon du site.

## Fichiers requis

### Logo
- `Logo.png` - Logo principal (format PNG recommandé)
- `Logo.jpg` - Alternative au PNG (optionnel)
- `Logo.svg` - Version vectorielle du logo (optionnel, utilisé comme fallback)

### Favicon
- `favicon.ico` - Favicon principal (format ICO)
- `favicon.png` - Alternative au ICO (optionnel)
- `favicon.svg` - Version vectorielle du favicon (optionnel)

## Instructions

1. Placez votre logo dans ce dossier avec le nom `Logo.png` (ou `Logo.jpg`, `Logo.svg`)
2. Placez votre favicon dans ce dossier avec le nom `favicon.ico` (ou `favicon.png`, `favicon.svg`)
3. Le site utilisera automatiquement ces fichiers

## Formats supportés

- **Logo** : PNG, JPG, SVG
- **Favicon** : ICO, PNG, SVG

## Dimensions recommandées

- **Logo** : 200x60px (ou proportionnel)
- **Favicon** : 32x32px ou 64x64px
