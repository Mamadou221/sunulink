# Images de Services

Ce dossier contient les images de fond pour la section "Nos Services" de la page d'accueil.

## Images requises

### 1. `import-export-bg.jpg` (ou .png, .webp)
- **Usage** : Image de fond pour la carte "Import-Export"
- **Thème** : Commerce, fruits secs, export, transformation
- **Recommandations** :
  - Format : JPG, PNG ou WebP
  - Dimensions recommandées : 1200x800px minimum
  - Poids : Optimisé pour le web (< 500KB)
  - Contenu : Fruits secs, produits transformés, commerce international

### 2. `immobilier-bg.jpg` (ou .png, .webp)
- **Usage** : Image de fond pour la carte "Immobilier"
- **Thème** : Villas, appartements, gestion locative, immobilier au Sénégal
- **Recommandations** :
  - Format : JPG, PNG ou WebP
  - Dimensions recommandées : 1200x800px minimum
  - Poids : Optimisé pour le web (< 500KB)
  - Contenu : Villas, appartements, architecture sénégalaise

## Notes techniques

- Les images seront utilisées avec `background-size: cover`
- Un overlay semi-transparent sera appliqué pour garantir la lisibilité du texte
- Compatible Light et Dark mode
- Responsive (mobile, tablette, desktop)
