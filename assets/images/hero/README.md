# Image Hero - Page d'accueil

## Fichier requis

**hero-home.jpeg** - Image visuelle pour la section Hero de la page d'accueil (affichée à droite du texte)

## Spécifications recommandées

- **Format** : JPEG (optimisé pour le web)
- **Dimensions** : 1200x800px minimum (ratio 3:2 recommandé)
- **Poids** : < 400 KB (optimisé)
- **Contenu** : 
  - Transformation de fruits secs
  - Produits alimentaires naturels et bio
  - Ambiance professionnelle, moderne et premium
  - Représentation du négoce et de la distribution
  - Image inspirant confiance et clarté

## Placement

Placez votre image dans ce dossier avec le nom exact : `hero-home.jpeg`

## Comportement

- **Desktop** : Image affichée à droite du texte (layout 2 colonnes)
- **Tablette** : Image affichée à droite du texte (layout adapté)
- **Mobile** : Image affichée en haut, texte en dessous (layout 1 colonne)

## Fallback

Si l'image n'est pas présente ou ne charge pas, elle sera masquée automatiquement et seul le texte sera affiché.
