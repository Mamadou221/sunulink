# Logos des Partenaires

Ce dossier contient les logos des partenaires de SunuLink Global.

## Logos des partenaires

Pour ajouter les logos des partenaires :

1. Placez les fichiers logos dans ce dossier avec les noms suivants :
   - `transhipping-logo.png` (Transhipping International)
   - `partner-2.png` (Partenaire 2)
   - `partner-3.png` (Partenaire 3)

2. Formats acceptés : `.png`, `.jpg`, `.jpeg`, `.svg`

3. Dimensions recommandées : 200x80px (ou ratio similaire)

4. Les logos doivent être adaptés pour le mode clair et sombre

5. Les logos doivent être :
   - Bien centrés
   - Respecter les proportions (pas étirés)
   - Avoir une taille cohérente entre eux
   - Rester lisibles en light mode et dark mode
