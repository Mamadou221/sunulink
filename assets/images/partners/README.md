# Logos des Partenaires

Ce dossier contient les logos des partenaires de SunuLink Global.

## Logo Transhipping International

Pour ajouter le logo de Transhipping International :

1. Placez le fichier logo dans ce dossier avec le nom : `transhipping-logo.png`
2. Formats acceptés : `.png`, `.jpg`, `.jpeg`, `.svg`
3. Dimensions recommandées : 200x80px (ou ratio similaire)
4. Le logo doit être adapté pour le mode clair et sombre

## Structure attendue

```
partners/
  ├── transhipping-logo.png
  └── README.md
```

Le code HTML est déjà configuré pour afficher automatiquement le logo une fois le fichier ajouté.
