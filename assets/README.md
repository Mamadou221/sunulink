# Assets - SunuLink Global

## Logo
⚠️ **IMPORTANT** : Placez votre fichier logo dans ce dossier avec le nom :
- **Logo.png** (recommandé)
- **Logo.svg** (si format vectoriel)
- **Logo.jpg** (si format JPG uniquement)

Le site utilisera automatiquement votre logo une fois placé ici.
Voir LOGO_INSTRUCTIONS.txt pour plus de détails.

## Favicon
- **favicon.svg** : Favicon du site (format SVG)

## Images pour la page Immobilier - Gestion Locative
Placez les images suivantes dans ce dossier :
- **GS1.jpg** : Image 1 du carrousel Gestion Locative
- **GS2.jpg** : Image 2 du carrousel Gestion Locative  
- **GS3.jpg** : Image 3 du carrousel Gestion Locative

Les images doivent être au format JPG ou PNG et avoir une résolution recommandée de 1200x800 pixels minimum.

## Remplacement des images
Pour remplacer les images :
1. Nommez vos fichiers exactement comme indiqué ci-dessus
2. Placez-les dans le dossier `assets/`
3. Les images seront automatiquement chargées par le site
